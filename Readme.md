# 🚀 GitHub Profile Analyzer

A modern JavaScript project that fetches and displays GitHub user data using the GitHub API.

🔗 Live Demo: https://abhinav05-bhatpude.github.io/Github-Analyzer/

---

![Project Screenshot](screenshot.png)

## ✨ Features

- Search GitHub users by username
- Display profile image and bio
- Show followers, following, and public repositories
- View top repositories with clickable links
- Loading state and error handling
- Responsive modern UI
- Smooth animations and hover effects

---

## ⚙️ How It Works

1. User enters a GitHub username
2. JavaScript sends request to GitHub API
3. User data is fetched asynchronously
4. Profile information is displayed dynamically
5. Repository links are generated automatically
