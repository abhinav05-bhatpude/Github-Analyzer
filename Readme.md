# 🚀 GitHub Profile Analyzer

## 📌 Overview
This project fetches GitHub user data using JavaScript and displays it in a clean UI.

---

## 🔥 Features
- Search GitHub users
- Display profile details
- Show repositories
- Error handling
- Loading state

---

## 🧠 Concepts Used
- Fetch API
- Async/Await
- Promises
- DOM Manipulation
- Error Handling

---

## ⚙️ How It Works
- User enters username
- API fetches data from GitHub
- Data is processed and displayed dynamically
